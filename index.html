    What it is

    The Counter App is a tool designed to keep track of the number of therapies and exams that each therapist performs daily. The application also provides workload measurement tooling as well as data visualization capabilities.

    Why it exists

    The Counter App was built to provide an easy and efficient way to keep track of the daily workload of each therapist. With this app, therapists can log the number of therapies and exams they perform each day, and the app will automatically calculate the total workload and display it in a simple and easy-to-read format.

    The workload measurement tool allows therapists to see how many therapies and exams they are projected to execute and are capable of completing. This data can be used to optimize time management and workload distribution, ensuring that each therapist is working at an optimal level.

    The data visualization capabilities of the Counter App allow for easy analysis of the workload data and work that has been already completed. The app can create graphs and charts to visually display this data, making it easier to identify patterns and trends.

    This application was intended as a tool to optimize therapist output and ensure that each therapist is working at a realistic and optimal level.
